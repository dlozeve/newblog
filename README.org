* Personal blog

** Setup

Uses [[https://developers.cloudflare.com/pages/get-started/direct-upload/][Cloudflare Pages direct upload]].

#+begin_src sh
npm install
npx wrangler login
npx wrangler pages project create
#+end_src

** Build and deploy

#+begin_src sh
cabal run site build
npx wrangler pages deploy _site
#+end_src
